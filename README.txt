Introduction
============

Quintagroup Captcha Core allows to configure captchas on your Plone site. 

After product installation you'll see 'Plone captchas setup' option under 
'Add-on Product Configuration' in Site Setup. There you can select what kind 
of captchas you want to appear: either static or dynamic. In case you select 
dynamic captchas you will be able to configure captchas look by using different 
font sizes, background and font colours, period, amplitude, random values.

Quintagroup Captcha Core does not add captcha to your Plone site, it only allows
its configuring. To protect your Plone forms with captcha - use 'quintagroup.plonecaptchas'
package.

Requirements
------------

* Plone 3.x
